# Serverless Tasks API
## Overview
In this project I will develop a Serverless Tasks API with typescript.

## Instalation & Running
Just run yarn to install dependencies:
```shell
yarn
```

To run the database:
```shell
yarn dynamo
```

and then:
```shell
yarn dev
```
to start development server and test the api
